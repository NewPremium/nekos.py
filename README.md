# nekos.py
Nekos API 를 사용한 파이썬 모듈

## Requirements
- Python 3.6 and up - https://www.python.org/downloads/

# Install
```
pip install nekos.py
```

# Usage example
```py
import nekos
print(nekos.cat())

>>> 'https://cdn.nekos.life/meow/04D5A.jpg'
```
